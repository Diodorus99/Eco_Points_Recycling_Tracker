# Eco_Points_Recycling_Tracker
An app that awards points to households for their recycling practices.
It keeps track of how much each household recycles, incrementing points for the recycling event based on total weight recycled and adding the recycling event to corrsponding household.
Stores a map of households with their Household ID as their key, for easy access. Prints all information available regarding the household. Prints relevant statistics,such as household with more points.
